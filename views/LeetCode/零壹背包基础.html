# 零壹背包基础

钟 波

# 题目描述

有N件物品和一个最多能背重量为W的背包。第i件物品的重量是weight[i]，得到的价值是value[i] 。 每件物品只能用一次 ，现在让你用这个背包装物品，最多能装的价值是多少？

示例1：

输入：N = 3, W = 4, weight = [3, 3, 4], value = [15, 20, 30]
输出：35

# 题解

很经典的一个动态规划问题。

1、定义dp数组

dp[i,j] ：当背包容量为j时，现有i件物品可以装，所能装得物品的最大价值。

2、确定状态转移方程

  • 如果装不下当前物品，那么装 前i个物品所得的最大价值 和 前i-1个物品所得的最大价值 是一样的。即有 dp[i,j] = dp[i-1][j]

  • 如果装得下当前物品：

    • 假设1：选择将当前物品装入背包，那么在给当前物品预留了相应得空间情况下，前i-1个物品所得的最大价值加上当前物品i的价值就是总价值。
    • 假设2：选择不装当前物品，那么装 前i个物品所得的最大价值 和 前i-1个物品所得的最大价值 是一样的。

    选取假设1和假设2中的 较大价值 作为当前的总价值即可。即 dp[i,j] = max(dp[i-1][j], dp[i-1][j-weight[i]]+value[i])

3、边界条件

dp[0,j] ：当前背包容量为1，没有物品可以装，所以最大价值为0，即 dp[0,j] = 0

dp[i,0] ：当前背包容量为0，可装前i个物品。由于背包没有容量，装不下任何物品，所以最大价值为0， 即dp[i,0] = 0

4、确定dp数组的计算顺序

先遍历物品，再遍历背包，也就是按层进行遍历。

代码如下：

public class MainTest {
    
    public static void knapsack(int[] weight, int[] value, int bagSize) {
        // 定义dp数组
        int[][] dp = new int[weight.length + 1][bagSize + 1];
        // 先遍历物品，再遍历背包容量
        for (int i = 1; i < weight.length + 1; i++) {
            for (int j = 1; j < bagSize + 1; j++) {
                // 装不下当前物品
                if (j < weight[i - 1]) {  
                    dp[i][j] = dp[i - 1][j];
                } else {  // 能装下，选择装与不装，取价值最大的。
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i - 1][j - weight[i - 1]] + value[i - 1]);
                }
            }
        }
        // 打印dp数组
        for (int[] ints : dp) {
            System.out.println(Arrays.toString(ints));
        }
        System.out.println("背包所获得的最大价值为：" + dp[weight.length][bagSize]);
    }
    
    public static void main(String[] args) {
        int[] weight = {1, 3, 4};
        int[] value = {15, 20, 30};
        int bagSize = 4;
        /*int[] weight = {2,3,4,5};
        int[] value = {3,4,5,8};
        int bagSize = 8;*/
        knapsack(weight, value, bagSize);
    }
}

很经典的一个动态规划问题。

1、定义dp数组

dp[j] ：当背包容量为j时，所能装得物品的最大价值。

2、确定状态转移方程

dp[j] 可以通过 dp[j - weight[j]] 推导出来，dp[j - weight[i]] 表示容量为 j - weight[i] 的背包所得的最大价值。

dp[j - weight[i]] + value[i] 表示 容量为 j - 物品i重量 的背包 加上 物品i的价值。

所以此时dp[j]有两个选择，一个是取自己dp[j]，一个是取dp[j - weight[i]] + value[i]，在这二者中取价值较大的即可。

于是得到状态转移方程为：

3、边界条件

当背包容量为0所背的物品的最大价值就是0，所以有 dp[0] = 0

dp数组在推导的时候一定是取价值最大的数，如果题目给的价值都是正整数那么非0下标都初始化为0就可以了，如果题目给的价值有负数，那么非0下标就要初始化为负无穷 。

这样才能保证dp数组在递归公式的过程中取的最大的价值，而不是被初始值覆盖了 。对于本题价值都是大于0的，因此初始化为0就可以了。

4、确定dp数组的计算顺序

先遍历物品，再遍历背包，也就是按层进行遍历。 注意这种方式背包的遍历顺序是不一样的，要从大往小遍历。 这与二维dp的实现是不一样的。

代码如下：

public class MainTest {
    
    public static void knapsack(int[] weight, int[] value, int bagSize) {
        int[] dp = new int[bagSize + 1];
        for (int i = 0; i < weight.length; i++) {
            for (int j = bagSize; j >= weight[i]; j--) {
                dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
            }
        }
        System.out.println("背包所获得的最大价值为：" + dp[bagSize]);
    }
    
    public static void main(String[] args) {
        int[] weight = {1, 3, 4};
        int[] value = {15, 20, 30};
        int bagSize = 4;
        /*int[] weight = {2,3,4,5};
        int[] value = {3,4,5,8};
        int bagSize = 8;*/
        knapsack(weight, value, bagSize);
    }
}
