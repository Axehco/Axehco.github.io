# 一维前缀和

# 简介

前缀和是指某序列的前n项和，可以把它理解为数学上的数列的前n项和，而差分可以看成前缀和的逆运算。合理的使用前缀和与差分，可以将某些复杂的问题简单化。差分将在后面再讲述。

前缀和算法有什么好处？

先来看一个例子：

输入一个长度为n的整数序列。接下来再输入m个询问，每个询问输入一对l、r。对于每个询问，输出原序列中从第l个数到第r个数的和。

针对这个问题，最容易想到的当然是 暴力求解 ，遍历区间求和。

这样的时间复杂度为 O(n*m) ，如果n和m的数据量稍微大一点就有可能超时，而我们如果使用 前缀和 的方法来做的话就能够将时间复杂度降到O(n+m)，大大提高了运算效率。

前缀和算法具体实现

  1. 首先做一个预处理，定义一个sum[]数组，sum[i]代表a数组中前i个数的和；
  2. 再求前缀和运算；
  3. 然后再执行查询操作，对于每次查询，只需执行sum[r]-sum[l-1]即可得到答案。

这样下来，时间复杂度为O(1)

前缀和原理

这样，对于每个询问，只需要执行 sum[r]-sum[l-1]。输出原序列中从第l个数到第r个数的和的时间复杂度变成了O(1)。这种方式叫一维前缀和 。

注意事项

  1. 为了统一公式 sum[l, r] = sum[r]-sum[l-1]，下标要从1开始。

  2. 将 sum[0] 定义为 0 ，避免下标的转换。

# 题目练习

题目传送门：AcWing 795. 前缀和 (opens new window)

AC代码：

#include <iostream>
#include <cstring>
#include <algorithm>

using namespace std;

const int N = 100010;
int n, m;
int a[N], s[N];

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++) scanf("%d", &a[i]);
    
    for (int i = 1; i <= n; i++) s[i] = s[i - 1] + a[i];
    
    while (m--) {
        int l, r;
        scanf("%d%d", &l, &r);
        printf("%d\n", s[r] - s[l - 1]);
    }
    return 0;
}

# 二维前缀和

# 简介

首先要会求一个矩阵每个元素(i, j)的左上方的元素之和；

比如 ：
i/j     1 2 3

1       3 7 1        3  10 11
2       2 4 0  -->   5  16 17  
3       9 4 2        14 29 32
得到此矩阵s

递推公式 s[i][j] = s[i-1][j] + s[i][j-1]-s[i-1][j-1]+matrix[i][j]; 
其中s[i-1][j-1]是s[i-1][j]和s[i][j-1]重叠计算的区域

那么求左上角为 (x1, y1)到右下角为 (x2, y2)的和：
s[x2][y2] - s[x1 - 1][y2] - s[x2][y1 - 1] + s[x1 - 1][y1 - 1]
其中s[x1-1][y1-1]是s[x1-1][y2]和s[x2][y1-1]重叠计算的区域

总结如下：

  1. s[i, j] 即为图中(左)红框中所有数的和：
s[i][j] = s[i - 1][j] + s[i][j - 1] - s[i - 1][j - 1] + a[i][j];
  1. (x1,y1),(x2,y2) 这一子矩阵中的所有数之和为：
s[x2][y2] - s[x1 - 1][y2] - s[x2][y1 - 1] + s[x1 - 1][y1 - 1]

# 题目练习

题目传送门：AcWing 796. 子矩阵的和 (opens new window)

AC代码：

#include <iostream>
#include <cstring>
#include <algorithm>

using namespace std;

const int N = 1010;

int n, m, q;
int s[N][N], a[N][N];

int main() {
    scanf("%d%d%d", &n, &m, &q);
    for (int i = 1; i <= n; i++)
        for (int j =  1; j <= m; j++)
            scanf("%d", &a[i][j]);

    for (int i = 1; i <= n; i++)
        for (int j =  1; j <= m; j++)
            s[i][j] = s[i - 1][j] + s[i][j - 1] - s[i - 1][j - 1] + a[i][j];

    while (q--) {
        int x1, y1, x2, y2;
        scanf("%d%d%d%d", &x1, &y1, &x2, &y2);
        printf("%d\n", s[x2][y2] - s[x1 - 1][y2] - s[x2][y1 - 1] + s[x1 - 1][y1 - 1]);
    }
    return 0;
}
